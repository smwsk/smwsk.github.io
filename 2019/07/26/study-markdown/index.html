MarkDown常用的语法
入门 MarkDown

标题

用#号来进行创建标题

# 一级标题
## ##二级标题
### ###三级标题

列表

使用 * - + 三种字符来进行展示列表

例子：
    * 列表1
    - 列表2
    + 列表3

效果

  • 列表1
  • 列表2
  • 列表3

代码块

使用``` 内容 ```的形式来表达

强调

在Markdown中，可以使用 * 和 _ 来表示斜体和加粗。

例子:
    **Coding，让开发更简单**
    __Coding，让开发更简单__

效果:

Coding，让开发更简单
Coding，让开发更简单

缩进

使用>符号进行缩进

例子:
使用>进行缩进

表格

普通表格

姓名|技能|排行
--|:--:|--:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟

效果:

姓名 技能 排行

姓名 | 技能 | 排行
:----------- | :-----------: | -----------:
刘备|哭|大哥
关羽|打|二哥

效果:

姓名 技能 排行
刘备 哭 大哥
关羽 打 二哥
张飞 骂 三弟

说明:
:--表示居左
:--: 表示居中
--: 表示居右

图片

使用![Alt text][id]的方式来进行设置图片信息 

例子:
    ![Alt text](/path/to/img.jpg)
    或
    ![Alt text](/path/to/img.jpg "Optional title")

修改图片大小

    ![test image size](url){:class="img-responsive"}
    ![test image size](url){:height="50%" width="50%"}
    ![test image size](url){:height="100px" width="400px"}

效果:

{: width=”100px” height=”100px”}
