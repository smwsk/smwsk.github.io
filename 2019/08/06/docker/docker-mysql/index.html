Docker安装Mysql
Docker Mysql
Docker

阅读次数:  

查找镜像

docker search mysql

下载镜像

docker pull mysql

查看安装的镜像

docker images

创建容器

docker run --name some-mysql -p 3307:3306 -e MYSQL_ROOT_PASSWORD=Wsk1234.. -e MYSQL_DATABASE=renren-fast -d mysql:latest

参数：

--name 给容器取个名称
-p 端口映射，把docker内部的3306端口映射到主机的3307端口
-e mysql环境参数 更详情的参考 https://hub.docker.com/_/mysql
-d 在后台运行
MYSQL_ROOT_PASSWORD 数据库密码
MYSQL_DATABASE 初始化一个数据库

远程连接

当前的新建出来的数据库还无法远程连接，要进行登录到mysql中进行修改配置

进入容器

docker exec -it 0abb bash

oabb 指的是容器的id(不用全写，只要前缀能唯一确定的mysql容器的id即可),容器id可以通过
docker images 进行查看

登录mysql

进入容器后，通过mysql命令进行登录后进行修改配置

root@0abbaf97c7d6:/# mysql -uroot -p

修改配置

mysql5.0版本用以下命令

GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'password' WITH GRANT OPTION;

mysql8.0版本用以下命令

use mysql;
update user set host='%' where user='root';
GRANT ALL ON *.* TO 'root'@'%';
ALTER USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY 'Zr12345..';
#语句生效
FLUSH PRIVILEGES;

用mysql8.0版本的时候，如果遇到一下情况

ERROR 1062 (23000): Duplicate entry '%-root' for key 'PRIMARY'

说明use表中的root用户有两条记录，删掉限权是本地的就可以了

delete from user where user='root' and host='localhost';
